# Solar System Simulation

A simulation of our solar system built with Python and Pygame, using real astronomical data and Newtonian physics.

---

## Features

- Real gravitational physics using Newton's law of universal gravitation
- Accurate planetary masses and orbital velocities
- Orbital path tracking for each planet
- Moon orbiting Earth
- Scales astronomical units (AU) to screen pixels

---

## Planets included

- Sun
- Earth (with Moon)
- Mercury

---

## Requirements

- Python 3.8 or higher
- Pygame

```
pip install pygame
```

---

## Usage

```bash
python "sol system simulation.py"
```

---

## Physics

The simulation uses real values:

| Constant | Value |
|----------|-------|
| Gravitational constant (G) | 6.67428e-11 |
| 1 AU | 149,600,000 km |
| Timestep | 1 day per frame |
